# Starter Kit Configurator Controls

The Starter Kit now uses three layers of variation:

1. **Profile** — the business being previewed.
2. **Theme** — the broad industry style attached to that profile.
3. **Configurator controls** — visitor-facing presentation options layered on top.

Current configurator controls include:

- Grid density: 2 / 3 / 4 columns
- Card style: soft / sharp / elevated
- Navigation style: top/default / pill / underline / boxed / side-style / compact / condensed
- Spacing: compact / balanced / spacious
- Background: clean / paper / wash
- Font mood: system / serif headings / compact
- Button style: rounded / square / bold
- Hero layout: split / centred / compact
- Image treatment: framed / full-bleed / minimal
- Section rhythm: alternating / plain / highlighted / collapsible
- CTA tone: direct / soft / campaign

This is not intended to become a full website builder. It is a proof layer showing how ASR can adapt the same maintainable foundation for different small-business needs and visual directions.

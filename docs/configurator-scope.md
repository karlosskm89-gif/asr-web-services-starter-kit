# Configurator Scope

The Starter Kit configurator is a proof layer, not a full website builder.

It demonstrates how the same maintainable website foundation can vary by industry example/profile, theme family, grid density, cards, navigation treatment, spacing, background, font mood, buttons, hero layout, image treatment, section rhythm, collapsible section presentation and CTA tone.

Notes:

- `nav=condensed` is a compact visual navigation treatment, not a JavaScript-powered menu drawer.
- `sections=collapsible` demonstrates collapsible-style presentation for showcase/profile sections and selected starter-kit sections.
- Full production sites should choose the most suitable pattern rather than exposing every option to public users.

# Public Readiness Notes

This public version has been prepared for GitHub/demo use.

## Excluded from public package

- `.env` files
- archive folders
- ZIP files
- duplicate working copies
- local-only working material

## Safety rules

Before publishing any future version, check for:

- API keys
- database URLs
- passwords or session secrets
- real client contact details
- private client names where permission has not been granted
- real form submissions or user data
- private images or brand assets not cleared for public use

## Recommended use

Use this repository as a technical showcase and live demo support asset. Client case studies and branded proposal mockups should remain separate unless they have been explicitly sanitised or cleared for public use.

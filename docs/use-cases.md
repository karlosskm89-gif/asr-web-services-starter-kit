# Use Cases

The starter kit can support several practical public/demo uses.

## 1. Small-business starter sites

A reusable base for websites that need:

- homepage
- services
- portfolio/projects
- testimonials
- FAQ
- contact/enquiry flow
- SEO metadata

## 2. Proposal mockups

The profile system makes it possible to prepare a niche-specific first draft quickly without rebuilding the route/view structure every time.

Examples:

- trades business
- café/restaurant
- private practice
- creator/personal brand
- community organisation

## 3. Public portfolio proof

This repo gives public evidence of:

- Express/EJS structure
- example-driven content
- reusable view patterns
- contact form handling
- documentation habits
- safe separation between public examples and private client work

## 4. Agency conversations

The starter kit can be used to show agencies that ASR can build maintainable, documented, backend-ready systems rather than one-off static pages.

## 5. Client discovery

A fictional profile or showcase page can help a client understand possible structure before a full project is scoped.

## Not intended for

- publishing real client code
- becoming a multi-tenant SaaS product
- replacing proper client discovery
- presenting fictional examples as completed client work

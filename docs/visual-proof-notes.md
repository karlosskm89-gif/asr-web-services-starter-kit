# Visual Proof Notes

For future portfolio screenshots, prefer views that show business value rather than isolated interface states.

Recommended screenshots:
- Homepage showing the value proposition clearly.
- Services or industry examples showing how the foundation adapts to a real business.
- Enquiry/contact journey showing how a visitor can become a lead.

Avoid relying too heavily on:
- Mobile menu states.
- Empty forms without context.
- Generic navigation screenshots.

The Starter Kit should visually demonstrate foundation proof: a maintainable website base that helps small businesses explain services, build trust and generate enquiries.

# Screenshots

This folder is reserved for real browser screenshots used in the GitHub README and proposals.

The repository already includes SVG preview assets under `public/images/showcase/`, so screenshots are useful but not required for the app to make sense.

Recommended files to add later:

```text
homepage-desktop.png
homepage-mobile.png
starter-kit-desktop.png
showcase-desktop.png
showcase-oakline-detail.png
case-studies-desktop.png
contact-form.png
```

Before committing screenshots, confirm they do not show private client data, local paths, credentials, admin screens, or form submissions.
